# Code Formatting Rules

## Line Length
- **Maximum line length: 120 characters**
- When editing HTML, CSS, or JavaScript files, ensure no line exceeds 120 characters
- Break long lines appropriately:
  - For HTML: Break at attribute boundaries or after opening tags
  - For CSS: Break after properties or selectors
  - For JavaScript: Break at logical points (after commas, operators, etc.)

## Indentation
- Use **4 spaces** for indentation (no tabs)
- Maintain consistent indentation throughout the file

## File Formatting
- Always apply these rules when creating or modifying files
- Ensure proper formatting before committing changes
